## 🤨 접근
### 🧬 로직
- board[i] = j 이면 i행 j열에 퀸이 있음
- 판을 돌며 dfs를 수행한다
  - 종료조건: 만약 깊이와 N(판의 크기)가 같다면 멈춤 (경우의 수 O)
  - depth행의 모든 j에 대해 유망성이 있는지 판별
    - 유망성 판별
      - 같은 열에 퀸이 있는지
      - 대각선에 퀸이 있는지
  - 유망성이 있다면 depth+1 탐색 시작
