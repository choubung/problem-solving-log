## 🤨 접근
### 🧬 로직
- 암호문의 길이를 입력받는다
- 암호문 길이만큼 원본 암호문을 각각의 요소로 입력받는다
- 명령어의 개수를 입력받는다
- 명령어의 개수만큼 반복한다
    - 명령어 종류를 입력받는다
    - 명령어 적용 시작 지점(x)를 입력받는다
    - 명령어 적용 개수(y)를 입력받는다
        - 만약 명령어가 D이면
            - x부터 y개만큼 지우기
        - 만약 명령어가 I이면
            - x+0부터 x+y까지 숫자 삽입하기
        - 만약 명령어가 A이면
            -  맨 뒤에 y개의 숫자 삽입하기
- 결과를 출력한다
