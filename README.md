# ✨ problem-solving-log
문제 해결(Problem Solving) 능력을 기르기 위한 개인적인 학습 기록 저장소입니다.
<br><br>

## 👩‍💻 주로 사용하는 언어
![Java](https://img.shields.io/badge/Java-E96420?style=flat-square&logo=java&logoColor=white)
<br><br>

## 🗂️ 디렉토리 구조
- 각 문제의 풀이는 플랫폼과 난이도별로 분류하여 관리합니다.
  - **SWEA**: SW Expert Academy
  - **Programmers**: 프로그래머스
- 문제 폴더 안에는 해답 코드와 해결 과정 아카이빙 README가 포함되어 있습니다.
```
problem-solving-log/
├── SWEA/
│ ├── D2/
│ │ └── P...
│ │   ├── README.md (코드 해결 과정 아카이빙)
│ │   └── Solution.java (해답 코드) 
│ ├── D3/
│ └── D4/
└── Programmers/
```
