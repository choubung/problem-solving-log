## 🤨 접근
### 🧬 로직
- 보드 판의 좌표의 합이 짝수이면 → 첫 칸과 같아야 하고
- 홀수이면 → 첫 칸과 달라야 함
- 내부에 나타날 수 있는 모든 8*8 보드판에 대해 위 조건으로 완전탐색하고, 최소값 찾기

<br>

## 🤯 실수
### 입력 실수
- 한 줄에 n과 m이 같이 입력되는데, 따로 입력된다고 생각함

### 잘못된 기준
- 탐색할 8*8의 처음 칸을 ‘옳은 기준’으로 설정했으나, 실제로는 처음 칸만 바꾸면 보드판이 완성되는 극단적인 반례가 존재할 수 있음
  → 첫 시작을 ‘W’로 고정해두고, 바꿔야하는 수를 구한 뒤 ‘B’로 시작하는 상황과 비교
  <br>
